# Retail Transactions Analysis

## Project Overview

This project performs Data Cleaning,Exploratory Data Analysis (EDA),and Dashboard Development on a Retail Transactions Dataset containing 1 Million transactions from 2020 to 2024.

The objective of this project is to derive business insights regarding customer behavior,sales trends,promotions,payment methods,and store performance.

---

## Dataset Information

- Total Records: 1,000,000
- Total Features: 13
- Time Period: 2020 - 2024
- Dataset Link:https://www.kaggle.com/datasets/prasad22/retail-transactions-dataset/data

### Features

- Transaction_ID
- Date
- Customer_Name
- Product
- Total_Items
- Total_Cost
- Payment_Method
- City
- Store_Type
- Discount_Applied
- Customer_Category
- Season
- Promotion

## Key Insights

- Average transaction cost is $52.46.
- Average basket size is 5.5 items.
- Discounts are applied in nearly 50% of transactions.
- Payment methods are evenly distributed.
- Sales remain stable across seasons.
- Store traffic is balanced across all store types.
